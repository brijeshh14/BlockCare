BlockCare:
📌 Overview

BlockCare is a decentralized healthcare data management system built on the Internet Computer Protocol (ICP). It provides secure, tamper-proof, and patient-owned medical records with AES encryption, Internet Identity (DID) authentication, and Role-Based Access Control (RBAC).

✨ Key Features

🔐 Patient-Owned Records – Full control over medical data.

🛡️ End-to-End Security – AES encryption with blockchain audit trails.

🌐 Seamless Sharing – Interoperable access across hospitals and doctors.

⚡ Efficient Storage – IPFS used for encrypted off-chain medical files.

🔧 Tech Stack

Blockchain: Internet Computer Protocol (ICP)

Storage: InterPlanetary File System (IPFS)

Encryption: AES (Client-Side)

Authentication: Internet Identity (DID)

Access Control: RBAC (Doctors, Patients, Admins)

🏥 Impact

Protects patient records from cyberattacks.

Eliminates data silos and reduces treatment delays.

Builds trust with transparent and tamper-proof records.
